## 📚 Assignment: Building REST APIs with FastAPI

### 🎯 Objective
Learn how to build and deploy RESTful APIs using the FastAPI framework in Python. Practice designing endpoints, handling requests, and returning JSON responses.

### 📝 Tasks

#### 🛠️ Task 1: FastAPI Setup
**Description:**
Set up a basic FastAPI application and run a simple server.

**Requirements:**
- Install FastAPI and Uvicorn.
- Create a FastAPI app instance.
- Start the server and access a basic endpoint.

#### 🛠️ Task 2: Create Endpoints
**Description:**
Design multiple REST API endpoints for basic CRUD operations.

**Requirements:**
- Implement GET, POST, PUT, and DELETE endpoints.
- Handle JSON request and response bodies.
- Return appropriate status codes and messages.

#### 🛠️ Task 3: Data Handling
**Description:**
Store and manage data using Python lists or dictionaries.

**Requirements:**
- Use in-memory data structures to simulate a database.
- Allow users to create, read, update, and delete items.
- Validate input and handle errors gracefully.

#### 🛠️ Task 4: Bonus - API Documentation
**Description:**
Explore FastAPI's automatic API documentation features.

**Requirements:**
- Access the interactive Swagger UI and ReDoc documentation.
- Test endpoints using the built-in docs.
