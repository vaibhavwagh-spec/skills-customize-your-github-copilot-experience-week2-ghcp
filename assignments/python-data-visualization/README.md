## 📚 Assignment: Python Data Visualization

### 🎯 Objective
Learn to create charts and graphs in Python using matplotlib or plotly. Practice visualizing data and customizing plots for clear communication.

### 📝 Tasks

#### 🛠️ Task 1: Setup & Basic Plot
**Description:**
Install matplotlib or plotly and create a simple line chart from sample data.

**Requirements:**
- Install matplotlib or plotly.
- Create a basic line chart from a list of numbers.
- Label axes and add a title.

#### 🛠️ Task 2: Bar & Pie Charts
**Description:**
Visualize categorical data using bar and pie charts.

**Requirements:**
- Create a bar chart for category counts.
- Create a pie chart for proportions.
- Customize colors and add legends.

#### 🛠️ Task 3: Data from File
**Description:**
Read data from a CSV file and plot it.

**Requirements:**
- Load data from a CSV file using pandas.
- Plot relevant columns as a chart (line, bar, or scatter).
- Save the plot as an image file.

#### 🛠️ Task 4: Bonus - Interactive Plots
**Description:**
Use plotly to create interactive charts.

**Requirements:**
- Create an interactive plot (hover, zoom, etc.).
- Export the plot as HTML.
