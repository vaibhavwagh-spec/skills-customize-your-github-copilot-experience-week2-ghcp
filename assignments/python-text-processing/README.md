## 📚 Assignment: Python Text Processing

### 🎯 Objective
Practice working with strings, file input/output, and text manipulation in Python. Learn to read, process, and transform text data from files.

### 📝 Tasks

#### 🛠️ Task 1: Reading Files
**Description:**
Write a Python program to read text from a file and display its contents.

**Requirements:**
- Open and read a text file using Python.
- Print the file contents to the console.
- Handle file not found errors gracefully.

#### 🛠️ Task 2: String Manipulation
**Description:**
Process the text data by performing various string operations.

**Requirements:**
- Count the number of words and lines in the file.
- Convert all text to lowercase and remove punctuation.
- Display the processed text and statistics.

#### 🛠️ Task 3: Writing Output
**Description:**
Save the processed text and statistics to a new output file.

**Requirements:**
- Write the cleaned text and statistics to a new file.
- Ensure the output file is properly formatted.
- Handle file writing errors gracefully.

#### 🛠️ Task 4: Bonus - Advanced Text Features
**Description:**
Add extra features like finding the most common word or searching for keywords.

**Requirements:**
- Identify and display the most frequent word in the text.
- Allow the user to search for a specific keyword and show its occurrences.
